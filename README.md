# Dotfiles
This directory contains the dotfiles for my system

## Requirements
Ensure you have the following installed on your system

### Git
```
pacman -S git
```

### Stow
```
pacman -S stow
```

## Installation
```
$ git clone git@github.com/haschad/dotfiles.git
$ cd dotfiles
$ stow .
```
